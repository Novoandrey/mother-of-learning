# [draft] Роли и клиенты

> Заглушка. Содержание будет наполняться постепенно.

В проде роли: `owner` / `dm` / `player`. Несколько `dm`-аккаунтов на одну кампанию уже поддерживаются — это фундамент для west-marches со множеством DM. На горизонте: `spectator` (наблюдатель без управляемых персонажей) для live-broadcast, и опциональный флаг `see_all` для расширенного доступа к скрытым событиям.

- Таблица ролей: что видит и что может каждая
- Как создаются и отзываются роли (приглашения, owner-флоу)

- Партии — UI-агрегация, не security-граница
