Главный принцип: Mother of Learning — инструмент для DM и игроков, не симулятор мира. Симуляция возникает там, где снимает с DM рутину (бросить кубик, посчитать стоимость в тиках). Симуляция не возникает там, где ограничивает DM (автономные NPC с собственной волей, мир, который «идёт сам»).
