# Петля как ядро

> Заглушка. Содержание будет наполняться постепенно.

Mother of Learning — игра про прогрессию через циклы. Loop = период игрового времени с фиксированным началом и условием сброса; внутри — дни (или тики целевой модели), к которым привязаны все события. Все ноды и транзакции получают `(loop, day)` координаты — это позволяет перемотку, фильтрацию ленты по петле, прогрессбар «где сейчас пачки».

## Что планируется в статье

- Что такое петля в нарративе vs в коде
- Frontier — фронт текущей петли по дню
- Как сессии крепятся к диапазону дней
- Прогрессбар петли и наложения сессий
- Базис для time-travel UI и hex-crawl таймингов
