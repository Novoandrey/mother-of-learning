# [draft] Общак и складчина

> Заглушка. Содержание будет наполняться постепенно.

Общак (stash) — одна нода на кампанию: общий кошелёк + общий инвентарь, на который скидываются деньги и предметы. Кнопки «Положить в общак» / «Взять из общака» в actor bar. Если у PC не хватает на расход — форма предлагает добрать из общака. Складчина — групповая покупка: несколько PC скидываются на общую вещь, которая ложится в общак, через атомарные парные переводы.

## Что планируется в статье

- Stash-нода: схема, RLS, как сидится
- Put / Take — UX и под капотом
- Shortfall prompt: rich/poor/empty модусы
- Складчина: form + распределение долей
- InventoryGrid → InventoryTab: общая компонента для PC и stash
