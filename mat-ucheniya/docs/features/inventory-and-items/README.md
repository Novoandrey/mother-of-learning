# [draft] Инвентарь и предметы

> Заглушка. Содержание будет наполняться постепенно.

База предметов: 1118 в `mat-ucheniya` (274 hand-curated SRD + 844 из dnd.su). У каждого: категория, редкость, цена, вес, слот, источник, доступность, флаг attunement. Инвентарь PC — вкладка `?tab=inventory` на странице персонажа. Инвентарь стаха — таб «Предметы» на странице общака. Группировка и сортировка через URL-параметры.

## Что планируется в статье

- Создание предмета: категория, цена, attunement
- Item permalink: страница `/items/[id]` с историей транзакций
- InventoryTab: фильтры по петле и дню, группировка
- Settings: категории, слоты, источники, доступность
- Связь предмета с транзакциями через `item_node_id` FK
