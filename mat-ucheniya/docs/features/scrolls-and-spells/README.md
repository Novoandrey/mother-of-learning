## Критические грабли (набито этой сессией — не наступи)

1. **PostgREST embed-ambiguity = молчаливый пустой каталог.** Второй FK на
   `nodes` из дочерней таблицы (у нас `item_attributes.schema_for_node_id`)
   сделал ВСЕ эмбеды `nodes → child!inner(...)` неоднозначными → пустые
   каталог/покупка/крафт, БЕЗ ошибки (код делал `const { data }`, глотая
   error). Если связываешь `spell`/`scroll` с нодами через FK — **не плоди
   второй FK на nodes из одной таблицы**; предпочитай soft-link (uuid-колонка
   без FK) + отдельный резолв-запрос. После DDL-смены схемы **обязательно**
   `notify pgrst, 'reload schema'` (иначе PostgREST держит старую топологию).
   См. `MEMORY.md → postgrest-embed-fk-ambiguity`.
2. **Числа механики = ДМ-настройки, НЕ хардкод** (AGENTS.md). Таблица цен
   свитков, `50×уровень`, `2ч×уровень`, hoursPerDay/Week, maxSpellLevel-
   формула(правило, не настройка) → `scribe_settings`/`spell_settings` в
   `campaigns.settings`, parse-хелпер с дефолтами (паттерн
   `parseCraftSettings`). Числа из спеки = **дефолты парсера**.
3. **Read-хелперы должны падать ГРОМКО.** `const { data } = await …` глотает
   ошибку → пустой список маскирует сбой под «нет данных» (потеряли час на
   пустых схемах). Бери `{ data, error }` и `if (error) throw error` — экран
   ловит и показывает ошибку.
4. **Sidebar cache** (AGENTS.md): вставка/правка `node_types`/`nodes` (новые
   типы `spell`/`scroll`, сид спеллов) трогает `nodes` → инвалидируй
   (`invalidateSidebar`/`invalidateSidebarRemote` для CLI-сида).
5. **Накат миграций** (`MEMORY.md → db-migration-apply`): сам катишь на прод —
   `ssh andrey@37.27.254.49 "docker exec -i supabase-db psql -U supabase_admin
   -d postgres -v ON_ERROR_STOP=1" < …/NNN.sql`. Verify-блок обязателен
   (стиль 124–129). После schema-DDL — reload PostgREST (см. грабли №1).
6. **/tg нельзя превьюить в браузере** (нужна Telegram initData). Верификация:
   `npm run typecheck` + `npm run build` + `vitest` + агент **ux-auditor**
   (обязателен после любой /tg UI-правки и перед PR). Плюс ручной смоук
   Andrey.
7. **party_level уже = 9** (задано этой сессией). maxSpellLevel(9)=5-й круг.
