# [draft] Энкаунтеры

> Заглушка. Содержание будет наполняться постепенно.

Excel-style грид инициативы для боёвки D&D 5e. HP, AC, спасброски, условия, эффекты, лог боя. Игрок может бросать атаки и фиксировать урон по своему PC; DM правит всех. Status gate: только activе энкаунтеры показывают actions; в read-only — просмотр лога. Лут энкаунтера через автогенерацию (см. бухгалтерия).

## Что планируется в статье

- Создание энкаунтера: участники (PC + monsters), порядок инициативы
- Pill-редактор условий и эффектов
- Death saves и AC
- Лог боя и хронология ходов
- Mirror-нода энкаунтера в каталоге (зачем нужна)
- Автогенерация лута после энкаунтера
