# Монстры и статблоки

> Заглушка. Содержание будет наполняться постепенно.

База монстров с SRD-сидом + homebrew. У каждого монстра — стандартный D&D 5e статблок: HP, AC, скорости, спасы, навыки, иммунитеты/сопротивления, атаки, способности. Подключаются в энкаунтер как participant'ы с независимыми HP-pool'ами для каждого экземпляра. На горизонте — Phase B structured abilities extraction для encounter assistant (IDEA-056).

## Что планируется в статье

- SRD seed: что входит, как пополняется
- Homebrew монстры: создание, статблок-форма
- Подключение в энкаунтер: legendary, реакции, multi-instance
- Миграции 013–014, 018–020, 023 — что в каждой
