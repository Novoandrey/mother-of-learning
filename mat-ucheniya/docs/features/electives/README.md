# Факультативы

> Заглушка. Содержание будет наполняться постепенно.

Факультатив — нода типа `elective`, привязанная к одной или нескольким PC через ребро `has_elective`. Используется для трекинга внеосновных активностей (тренировка, изучение языка, социалка). На горизонте — преобразование в бонусы к статам PC (IDEA-037).

## Что планируется в статье

- Создание факультатива и привязка к PC
- Что отличает elective от обычной ноды
- Сидинг electives для всех кампаний (миграция 110)
- Будущее: бонусы к статам через electives
