# Стартовый сетап петли

> Заглушка. Содержание будет наполняться постепенно.

Подфича: при старте новой петли применяется campaign-level loan + stash seed + per-PC starting wallet. Все эти транзакции помечаются `autogen` — так приложение знает, что они сгенерированы и могут быть пере-применены. Если DM «тронул рукой» autogen-транзакцию — следующее «Применить» её обойдёт.

## Что планируется в статье

- Settings page: campaign loan + stash seed coins + stash seed items
- Per-PC настройка: starting wallet, takesStartingLoan
- Apply: что происходит при нажатии и что значит «обойти тронутые»
- Баннер «Стартовый сетап не применён» — когда показывается
- Spec-019 starter-overview: одностраничный DM-инструмент
