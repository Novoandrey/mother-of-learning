# Бухгалтерия

> Заглушка. Содержание будет наполняться постепенно.

Top-level раздел `/accounting/`: лента транзакций кампании, кошельки PC, общак, очередь заявок. Четыре кнопки в actor bar: + Доход / − Расход / + Предмет / − Предмет. Транзакции — append-only лог, баланс через `SUM()`. Парные переводы между PC связаны `transfer_group_id`. Категории редактируются DM и сидятся per-кампания.

## Что планируется в статье

- Виды транзакций: money / item / transfer
- Кошелёк PC и блок Wallet на странице персонажа
- Лента и фильтры (сворачиваемая панель, URL — single source of truth)
- Категории и как их настроить
- Что такое actor bar и как переключать персонажей
