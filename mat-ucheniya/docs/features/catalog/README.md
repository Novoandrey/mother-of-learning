# [draft] Каталог сущностей

> Заглушка. Содержание будет наполняться постепенно.

Главная вкладка приложения. Граф всех нод кампании: PC, NPC, локации, петли, сессии, энкаунтеры, факультативы, предметы. Текстовый поиск, фильтры по типу, сортировки, плоская навигация — никакой жёсткой иерархии папок, всё связано рёбрами.

## Что планируется в статье

- Типы нод и какие у каждой обязательные поля
- Поиск (как работает, по каким полям ищет)
- Фильтры и сортировки
- Создание новой ноды через каталог vs прямое создание
- Сайдбар: как сгруппированы ноды по типу
