# Player mobile — мобильный режим игрока

> Заглушка. Содержание будет наполняться постепенно.

Большая фича / серия спек. Игрок получает mobile-first UI поверх того же датасета: лист персонажа со статами, инвентарём, спеллами; история «что было с моим персонажем»; лента летописи; голосование/просмотр квестов; кошелёк и транзакции. Десктоп-DM не трогаем — у DM остаётся плотный панельный UI.

## Что планируется в статье

- Лист персонажа: layout, разделы, редактирование
- История PC: фильтр ленты по нему, петлевой timeline
- Чтение летописи на мобиле
- Кошелёк и транзакции — мобильная форма
- Скоп: что входит в первую спеку, что в следующие
- Не входит: грид энкаунтера (десктоп-only остаётся)
