# Дизайн-токены

> Заглушка. Содержание будет наполняться постепенно.

`mat-ucheniya/STYLE.md` — single source of truth для всех UI стилей. Inputs: rounded-lg + border-gray-200 + focus blue. Buttons: primary blue, secondary gray, compact-варианты для headers. Chips: amber/emerald/red/gray для статусов. Контрасты WCAG AAA для всех ledger-цветов. Принцип «реюзай, не городи» — если className не матчится с токеном, остановись и сверься.

## Что планируется в статье

- Inputs, textareas, selects
- Buttons (primary/secondary, full/compact, text-link)
- Chips и status colors
- Контрасты и accessibility
- Как добавить новый токен (когда оправдано)
