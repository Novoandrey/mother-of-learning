# Мать Учения

Бесплатный open-source инструмент для ведения настольных ролевых кампаний.
Граф сущностей с перекрёстными ссылками — по духу ближе к Obsidian, чем к красивому приложению.

Разрабатывается на примере сложной DnD 5e кампании по мотивам "Mother of Learning".

## Что умеет (v0.1)

- Каталог сущностей (НПС, персонажи, локации, группы, организации...)
- Поиск по имени и описанию (кириллица, регистронезависимый)
- Фильтрация по типу сущности
- Карточка сущности со всеми полями и тегами
- Связи между сущностями (направленные рёбра графа)
- Навигация по связям в один клик
- Создание новых сущностей и связей

## Стек

- **Frontend**: Next.js + React + Tailwind CSS
- **Backend**: Supabase (Postgres + Auth + Realtime)
- **Деплой**: Vercel + Supabase Cloud

## Запуск локально

```bash
cd mat-ucheniya
cp .env.local.example .env.local
# Заполни .env.local значениями из Supabase Dashboard → Settings → API
npm install
npm run dev
```

## База данных

Миграции и seed-данные лежат в `supabase/`:

- `migrations/001_initial_schema.sql` — таблицы, индексы, триггеры
- `seed.sql` — тестовые данные из реальной кампании (21 сущность, 20+ связей)

Применить через SQL Editor в Supabase Dashboard.
