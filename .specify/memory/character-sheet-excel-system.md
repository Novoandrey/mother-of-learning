# Лист персонажа — Как работает таблица "Персонажи"

> Документация системы автоматизации Excel-таблицы "Персонажи_примеры.xlsx".
> Источник: таблица + описание от ДМа (2026-04-15).
> Для будущего спека листа персонажа (IDEA-017) и Zettelkasten.

## Общая структура

Каждый персонаж — **отдельный лист** в xlsx-файле.
Примеры: Маркус (монах 8), Бальтазар (бард 8), Дамбиниус (клерик 6).

### Расположение данных

| Область | Колонки | Что хранит |
|---------|---------|------------|
| Шапка | A1–H1 | Имя, класс, уровень |
| Характеристики | A3:D8 | 6 статов (Сл, Лвк, Тл, Инт, Мдр, Хар) + модификаторы + спасброски |
| Боевые | E3:G7 | КД, инициатива, скорость, макс/тек/врем ХП, кость ХП |
| Мастерство | B9:C9 | Бонус мастерства (автоматически) |
| Навыки | A10:D27 | 18 навыков с привязкой к стату, бонус мастерства (0/0.5/1/2) |
| Снаряжение | H2:H9+ | Список предметов |
| Умения | K2:M27 | Расовые, классовые, фиты — с источником |
| Заклинания | N13:S19+ | Круг, заклинание, источник (2 колонки) |
| Ресурсы | N2:Q9 | Кость БИ, ци, ячейки заклинаний — всего/осталось |
| Валюта | E24:G29 | Медь, серебро, золото, платина |
| Оружие | B30:B32 | Владения оружием |
| Магазин | A35:B42 | Предметы с ценами (для покупок) |

## Система автоматизации (формулы)

### 1. Бонус мастерства (от уровня)
```
C9 = FLOOR((Уровень - 1) / 4, 1) + 2
```
- Уровень 1–4 → +2
- Уровень 5–8 → +3
- Уровень 9–12 → +4
- и т.д.

### 2. Модификатор характеристики (от значения стата)
```
Модификатор = FLOOR((Стат - 10) / 2, 1)
```
Примеры:
- Стат 8 → мод −1
- Стат 10 → мод 0
- Стат 14 → мод +2
- Стат 18 → мод +4
- Стат 20 → мод +5

Формула одинаковая для всех 6 характеристик:
```
C3 = FLOOR((B3-10)/2, 1)   # Сила
C4 = FLOOR((B4-10)/2, 1)   # Ловкость
C5 = FLOOR((B5-10)/2, 1)   # Телосложение
C6 = FLOOR((B6-10)/2, 1)   # Интеллект
C7 = FLOOR((B7-10)/2, 1)   # Мудрость
C8 = FLOOR((B8-10)/2, 1)   # Харизма
```

### 3. Проверки навыков (модификатор + мастерство)
```
Навык = МодСтата + FLOOR(БонусМастерства × Множитель, 1)
```
Где Множитель (колонка D):
- **0** — нет мастерства (только модификатор стата)
- **0.5** — половинное мастерство (бард, Мастер на все руки — округление вниз)
- **1** — мастерство (обычный навык с галочкой)
- **2** — экспертиза (удвоенный бонус мастерства)

Пример для Акробатики (привязана к ЛВК):
```
C10 = C4 + FLOOR($C$9 * D10, 1)
     = МодЛвк + FLOOR(Мастерство × Множитель, 1)
```

Каждый навык привязан к конкретной характеристике:
- ЛВК: Акробатика, Лвк рук, Скрытность
- ИНТ: Анализ, Магия, Природа, Религия, История
- СИЛ: Атлетика
- МДР: Внимательность, Выживание, Медицина, Проницательность, Уход за животными
- ХАР: Выступление, Запугивание, Обман, Убеждение

### 4. Спасброски
```
Спасбросок = МодСтата + БонусМастерства (если владеешь)
```
Примеры:
```
D3 = C3 + C9   # Сила + мастерство (Маркус: монах владеет Сл и Лвк)
D4 = C4 + C9   # Ловкость + мастерство
D5 = C5         # Телосложение (без мастерства — просто мод)
```

Спасбросок DC (сложность для противника):
```
J1 = 8 + МодМудрости + БонусМастерства
```

### 5. Хиты (HP)
```
Макс ХП = (среднее по кости × уровень) + (МодТелосложения × уровень)
```
**Формула средних**: для d8 → 5 × уровень + мод_тл × уровень.
Если игрок бросает кубами, вписывает вручную.

Кость ХП (Hit Dice): количество = уровень, тип = классовая кость (d8/d10/d12).

### 6. Класс доспеха (КД / AC)
```
По умолчанию: КД = 10 + МодЛвк
Монах: КД = 10 + МодЛвк + МодМдр    # Маркус: =C4+10+C7
```
Формула меняется в зависимости от класса и доспехов:
- Без доспехов: 10 + Лвк
- Лёгкий: 11/12 + Лвк
- Средний: 13/14/15 + Лвк (макс +2)
- Тяжёлый: фиксированный (15/16/17/18)
- Монах: 10 + Лвк + Мдр
- Варвар: 10 + Лвк + Тл
- Природный доспех (раса): обычно 13 + Лвк

### 7. Инициатива
```
По умолчанию: Инициатива = МодЛвк
```
Может добавляться бонус от черт (Alert, Remarkable Athlete и т.д.).
Формула: `F3 = C4 + C9` (Маркус: Лвк + мастерство от класса).

### 8. Попадание (Attack Bonus)
```
По умолчанию: Попадание = МодСилы + БонусМастерства
```
Для дальнобойных/ловких: МодЛвк + БонусМастерства.
Формула меняется в зависимости от оружия.
`E9 = C4 + C9` (Маркус: Лвк + мастерство, т.к. монах использует Лвк).

### 9. Ресурсы класса

Каждый класс имеет свой пул ресурсов:
- **Монах**: Ци (P3 = Уровень + МодМдр). Столбец "всего / осталось"
- **Бард**: Вдохновение барда (R3/S3). Ячейки заклинаний (P/Q по кругам)
- **Клерик**: Канал божества (R/S). Ячейки заклинаний (P/Q по кругам)

### 10. Скорость
```
Монах: Скорость = 30 + ДопСкорость + 10 (Подвижный?)
ДопСкорость = 10 + FLOOR((Уровень - 2) / 4, 1) × 5
```

## Принцип автоматизации

Пользователь вводит вручную:
1. **Значения характеристик** (6 чисел: 8–20)
2. **Уровень** (одно число)
3. **Галочки мастерства навыков** (0 / 0.5 / 1 / 2)
4. **Текущие HP** (если получил урон)
5. **Остатки ресурсов** (ци, ячейки, вдохновение)
6. **Снаряжение и валюту** (свободный текст / числа)

Всё остальное вычисляется автоматически:
- Модификаторы ← статы
- Навыки ← мод + мастерство × множитель
- КД ← формула класса
- Инициатива ← мод Лвк (+ бонусы)
- Попадание ← мод стата + мастерство
- Спасброски ← мод + мастерство (если владеешь)
- Макс HP ← уровень + тл
- Бонус мастерства ← уровень

## Ключевой инсайт для реализации

Лист персонажа — это **реактивный граф зависимостей**:
```
Уровень → Бонус мастерства → Навыки, Спасброски, Попадание, Ци, Ресурсы
Стат → Модификатор → Навыки, КД, Инициатива, HP, Попадание, Спасбросок
```

Каждое поле — либо **ввод** (синий текст в Excel), либо **формула** (чёрный).
Формулы пересчитываются мгновенно при изменении ввода.

Это идеально ложится на **event sourcing** (конституция v3, принцип V):
- Каждое изменение стата = событие
- Текущий стейт = replay всех событий
- "Конструктор слоями" (IDEA-017) = добавление событий один за другим

## Структура для будущей реализации

Минимальный лист персонажа:
1. **Базовые данные**: имя, класс, раса, уровень, предыстория
2. **6 характеристик**: значение → автомод → автоспасбросок
3. **18 навыков**: привязка к стату + множитель мастерства
4. **Боевые**: КД (формула), инициатива (формула), HP (формула), попадание (формула)
5. **Ресурсы**: ячейки заклинаний, ци, вдохновение — всего/осталось
6. **Снаряжение**: список строк
7. **Умения**: название + источник (раса/класс/фит)
8. **Заклинания**: круг + название + источник
